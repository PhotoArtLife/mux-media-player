<h1 align="center">Mux Media Player</h1>

<div align="center">
针对不同框架自定义自己的播放器（支持HTML5或Flash等）

## Versions

- [Vue版](./vue)
- [React版](./react)
- [Old版](./old)

## Features

- 一键截屏
- 屏幕控制播放与暂停，进度条自定义，播放与关闭，全屏按钮等
